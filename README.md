# SOC SSH Brute Force Detection Lab

## Overview
This project documents the detection, analysis, and response to a simulated SSH brute-force attack conducted in a controlled lab environment. The focus is on log analysis, attack characterization, and automated containment.

## Lab Environment
- Victim: Ubuntu Linux (SSH server)
- Attacker: Kali Linux
- Network: Bridged Adapter
- Logs Analyzed: /var/log/auth.log

## Key Skills Demonstrated
- SSH brute-force detection
- Linux authentication log analysis
- Attack pattern identification
- Incident reporting (SOC-style)
- Automated containment using security controls
